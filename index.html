Privacy Policy for Git Swipe

Last updated: February 2026

Git Swipe is a mobile application designed to help users browse and manage GitHub issues in a simplified interface.

Information We Collect

Git Swipe uses GitHub OAuth for authentication. When you log in, we may access basic public profile information provided by GitHub, such as:

  • GitHub username
  • Public profile information
  • Repository and issue access (based on granted permissions)

How We Use Information

The information retrieved via GitHub is used solely to:

  • Display GitHub issues
  • Allow interaction with repositories and issues
  • Improve user experience

Data Storage

Git Swipe does not store personal data on external servers. Any data accessed from GitHub is used in real time and may be stored locally on your device for performance purposes.

Third-Party Services

Git Swipe uses GitHub APIs. Use of GitHub services is subject to GitHub’s own privacy policy.

Security

We take reasonable measures to protect user data. However, no internet transmission is 100% secure.

Children's Privacy

Git Swipe is not intended for children under 13 years of age.

Changes to This Policy

This privacy policy may be updated periodically. Continued use of the app indicates acceptance of changes.
